事前準備

事前準備として、タイムを記録するためのデータベースを作成しておく必要があります。また、そのデータベースを使用する権限のあるユーザーとパスワードを確認しておいてください。データベース名には制約はありません。わかりやすい名前で作成してください。

sql.ymlの編集

プラグインロード時に生成されたMinecraftPフォルダ内にある、sql.ymlを編集します。sql.ymlの内容はデフォルトで以下のようになっています。

    Host:  localhost
    Port:  3306
    Database:  changethis
    User:  changethis
    Password:  changethis
    Enable:  changethis

spigotサーバーの動いているコンピュータ上でMySQLサーバーも動かしている場合はlocalhostのままで、もしも他のサーバーで動かしている場合はlocalhostを適切なホスト名に変更してください。PortはMySQLのデフォルトのポートのままであれば3306でOKです。データベース名、ユーザー名、パスワードをそれぞれ入力したことを確認したら、Enable:の値をtrueに書き換えて、リロードか再起動をしてください。うまくいくとコンソールに成功のメッセージが表示されます。
